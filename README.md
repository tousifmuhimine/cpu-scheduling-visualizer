# CPU Scheduling Visualizer

An interactive web-based CPU Scheduling Visualizer built to help students and developers understand how different CPU scheduling algorithms work internally through real-time simulation and visualization.

## 🚀 Features

- Visual simulation of CPU scheduling algorithms
- Interactive Gantt Chart generation
- Calculates:
  - Waiting Time
  - Turnaround Time
  - Completion Time
  - Average Waiting Time
  - Average Turnaround Time
- User-friendly interface
- Real-time process execution visualization
- Easy process input system

## 🧠 Implemented Algorithms

- First Come First Serve (FCFS)
- Shortest Job First (SJF)
- Shortest Remaining Time First (SRTF)
- Round Robin (RR)
- Priority Scheduling
- Longest Job First (LJF)
- Longest Remaining Time First (LRTF)

## 🛠️ Tech Stack

- HTML
- CSS
- JavaScript

## 📷 Project Purpose

CPU scheduling algorithms are one of the core concepts of Operating Systems, but understanding them theoretically can sometimes be confusing. This project aims to make learning easier by visually demonstrating how processes move and execute inside the CPU using different scheduling techniques.
